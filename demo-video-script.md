# RealEnglish 演示视频分镜脚本（Remotion · ≤3分钟）

> 比赛要求：演示视频总长不超过 3 分钟（180 秒）
> 本文档用于：(1) 指导录制各功能模块的屏幕素材 (2) 作为 Remotion 编码的提示词/剧本

---

## 一、视频全局参数

| 参数 | 值 |
|------|-----|
| 分辨率 | 1920 × 1080（16:9） |
| 帧率 | 30 fps |
| 总时长 | ≤ 170s（留 10s 余量） |
| 设计风格 | 多邻国风格 — 绿白主色调、大圆角、粗字体、活泼弹性动画 |

### 色彩规范（来自落地页 `frontend/index.html` CSS 变量）

```
主绿 #58CC02    深绿 #46A302    浅绿 #E5F8D1
白底 #FFFFFF    浅灰 #F7F7F7    边框 #E5E5E5
正文 #4B4B4B    标题 #2B2B2B    辅助字 #AFAFAF
蓝 #1CB0F6（A级）  紫 #CE82FF（B级）  橙 #FF9600（C级）
```

### 字体

- 英文/数字：**Plus Jakarta Sans**（800 粗体用于标题，600/700 用于正文）
- 中文：**PingFang SC** / Microsoft YaHei

## 二、素材录制清单

> **录制前准备**：启动项目（后端 + 前端），准备一个已注册且有一定学习记录的学生账号和一个教师账号。
>
> ```bash
> # 终端 1 — 后端
> cd backend && ../.venv/Scripts/python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
> # 终端 2 — 前端
> cd frontend && npm run dev
> # 浏览器打开 http://localhost:5173
> ```

| 编号 | 录制内容 | 来源 URL/组件 | 类型 | 录制时长 |
|------|---------|--------------|------|---------|
| A | 落地页：Hero → 功能卡片滚动 → 数据统计条 | `http://localhost:5173`（未登录）| 录屏 | ~15s |
| B | 注册弹窗 → 登录 → 学生仪表盘 → 定级测试 10 题 → 结果 | 落地页 modal → `LoginPage.vue` → `StudentDashboardPage.vue` → `PlacementTestPage.vue` → `PlacementResultPage.vue` | 录屏 | ~20s |
| C | 片段列表浏览 → 点击进入详情 → B站播放 → 中英对照 → 查词 → 悬浮助手 | `/student/clips` → `ClipDetailPage.vue` | 录屏 | ~25s |
| D | AI 出题 → 答题 → 即时反馈 → 错题本 → 单词本 | `PracticePage.vue` → `/student/wrong-note` → `/student/vocabulary`（或 `AccumulatePage.vue`） | 录屏 | ~25s |
| E | 口语练习：跟读模式 → 录音评测动画 → 评分展示 → 自由应答 → 情景对话 | `/student/speaking` → `/student/free-talk` | 录屏 | ~28s |
| F | 学生仪表盘（统计+图表+热力图）→ AI 学习助手对话 | `/student/dashboard` → `/student/assistant` | 录屏 | ~22s |
| G | 教师看板 → 学生管理 → 学生详情抽屉 → 师生聊天 → 教师助手 | `/teacher` → `/teacher/students` → `/teacher/messages` → `/teacher/assistant` | 录屏 | ~25s |

**录屏建议**：
- 使用 1920×1080 分辨率录制，Chrome 无痕窗口（无书签栏干扰）
- 每个素材录制时可以稍长一些，Remotion 中通过 `trim` 剪辑
- 鼠标移动平滑，点击前稍作停顿（方便后期加高亮圈）
- 避免录到系统通知弹窗

### 场景 0 — 开场 Logo（时间轴 0s – 8s，时长 8s）

**类型**：纯 Remotion 合成，无需录屏

#### 视觉

```
┌──────────────────────────────────────────────────────┐
│  背景：linear-gradient(180deg, #F2FCE9 0%, #FFF 100%) │
│                                                      │
│                    ┌──────────┐                       │
│                    │ learning │  ← SVG 居中偏右       │
│                    │  .svg    │     上下浮动动画       │
│                    └──────────┘     (float: 5s ease)  │
│                                                      │
│              RealEnglish                              │
│         AI 驱动的英语视听说学习平台                      │
│                                                      │
│       武汉理工大学 · 华工杯 AI 创新大赛                  │
└──────────────────────────────────────────────────────┘
```

#### 动画时序

| 时间 | 元素 | 动画 |
|------|------|------|
| 0.0s | 背景 | 已就位 |
| 0.3s | `learning.svg` | opacity 0→1，scale 0.8→1，spring 弹性 |
| 0.5s | SVG 浮动 | `translateY` 上下 ±16px，5s 循环 |
| 1.0s | "RealEnglish" 文字 | 从下方 40px 弹入（spring: damping=12, stiffness=200），颜色 `#58CC02`，字号 64px，字重 800 |
| 1.8s | 副标题 | opacity 0→1，translateY 10→0，字号 22px，`#4B4B4B` |
| 2.5s | 底部小字 | opacity 0→1，字号 14px，`#AFAFAF` |
| 5.0s | 整体 | 保持 2s 后准备转场 |

#### 转场

- 绿色圆形遮罩放大（`circle()` clip-path，从中心扩展到覆盖全屏），进入场景 1

#### Remotion 关键参数

```tsx
// 开场动画
const logoSpring = spring({ frame, fps: 30, config: { damping: 12, stiffness: 200 } });
const svgFloat = Math.sin(frame * 0.06) * 16;

// 转场
const circleReveal = interpolate(frame, [150, 180], [0, 150], { extrapolateRight: "clamp" });
```

### 关键动画参数

```tsx
// spring 弹性动画（用于文字/Logo入场）
const springConfig = { damping: 12, stiffness: 200 };

// 浮动动画（用于 SVG 装饰）
const floatY = (frame: number, speed = 0.06, amplitude = 16) =>
  Math.sin(frame * speed) * amplitude;

// 圆形遮罩转场
const circleProgress = interpolate(frame, [start, end], [0, 150], {
  extrapolateRight: "clamp",
});
// clipPath: `circle(${circleProgress}% at center)`

// 数字 count-up
const countValue = interpolate(frame, [start, end], [0, target], {
  extrapolateRight: "clamp",
});
// 显示：Math.round(countValue)

// 淡入 + 上移
const fadeUp = (frame: number, delay: number) => ({
  opacity: interpolate(frame - delay, [0, 15], [0, 1], { extrapolateRight: "clamp" }),
  transform: `translateY(${interpolate(frame - delay, [0, 15], [10, 0], { extrapolateRight: "clamp" })}px)`,
});
```

### 录屏素材引用

在 Remotion 中通过 `<OffthreadVideo>` 引用录屏素材：

```tsx
import { OffthreadVideo } from "remotion";

<OffthreadVideo
  src="/recordings/clip-a-landing.mp4"
  startFrom={0}           // 从素材的哪一秒开始
  endAt={15}              // 到素材的哪一秒结束
  style={{ borderRadius: 12 }}
/>
```

### 音频建议

- **BGM**：轻快、有节奏感的纯音乐（无版权），音量控制在 20-30%，不影响解说
- **音效**：转场时加轻快的 whoosh/click 音效，答题正确加叮咚音效，评分展示加计数音效
- **人声解说**（可选）：AI 配音或自行录制，语速适中（~150 字/分钟），与画面同步

## 五、素材录制详细指导

### 素材 A — 落地页（~15s）

1. 清除浏览器缓存/使用无痕窗口，打开 `http://localhost:5173`
2. 等待页面完全加载（SVG 插图出现、动画稳定）
3. 开始录制 → 停留 2s 在 Hero → 缓慢向下滚动 → 4 张功能卡片依次出现 → 继续滚动到绿色统计条 → 停止
4. **注意**：滚动速度均匀，不要来回滚动

### 素材 B — 注册+定级测试（~20s）

1. 使用无痕窗口，打开 `http://localhost:5173`
2. 点击"免费开始学习" → 弹窗中选择"我是学生" → 输入新用户名/邮箱/密码 → 点击注册
3. 自动登录跳转到 `/student/dashboard` → 停留 1s 展示仪表盘
4. 如果是新用户，看是否有定级测试引导 → 开始 10 题测试
5. 依次作答 10 题 → 注意展示题目+视频播放+反馈 → 结果页等级展示
6. **如果账号已有等级**：可以换一个新账号，或在已有账号上录制定级测试已完成的状态

### 素材 C — 影视片段学习（~25s）

1. 登录学生账号，进入 `/student/clips`
2. 展示片段卡片网格 → 悬停/点击分类筛选（展示分类下拉） → 点击难度筛选
3. 点击第一个片段进入详情 `/student/clips/1`
4. 点击播放按钮 → B站视频播放 3-4 秒
5. 滚动到对话文本区 → 点击"英文原文"→ 点击"大模型翻译"（展示切换和流式翻译）
6. 点击对话中的某个英文单词 → 查词卡片弹出 → 点击"收藏"
7. 点击右下角悬浮 AI 助手气泡 → 展示拖动

### 素材 D — AI 出题+答题（~25s）

1. 在片段详情页点击"AI 出题"
2. 等待加载（如果题目已存在会直接跳转） → `/student/practice?clip_id=1`
3. 展示题目 + 关联视频播放
4. 故意答错一题 → 展示红色反馈 → 再答对一题展示绿色反馈
5. 导航到 `/student/accumulate` → 展示错题本 Tab → 展示单词本 Tab → 点击复习模式

### 素材 E — 口语练习（~28s）

1. 导航到 `/student/speaking`
2. 展示三种模式标签 → 选择跟读模式
3. 播放原声 → 点击录音 → 展示录音动画和波形 → 等待评测结果
4. 展示评分面板（四个维度+总分）
5. 滚动到底部展示 ECharts 口语趋势图（如果无历史数据，展示空状态或模拟数据）
6. 切换到自由应答 → 展示 AI 问题 → 录音回答
7. 导航到 `/student/free-talk` → 展示场景卡片 → 选择场景和角色 → 发送一条文字消息 → 展示对话

### 素材 F — 仪表盘+AI助手（~22s）

1. 导航到 `/student/dashboard`
2. 展示顶部 3 统计卡片 → 展示 ECharts 折线图 → 展示热力图
3. 导航到 `/student/assistant`
4. 在聊天框输入"推荐适合我的学习片段" → 发送 → 等待 AI 回复
5. 展示 AI 的 Markdown 表格回复 + 工具调用时间轴

### 素材 G — 教师端（~25s）

1. 退出学生账号，登录教师账号（如 `teacher_wang` / W4X9 对应密码）
2. 自动跳转 `/teacher` 看板 → 展示 4 统计卡片 + 图表 + 预警条
3. 点击预警条 → 跳转学生管理 `/teacher/students`
4. 在学生列表页展示搜索、等级更新
5. 点击某个学生行 → 右侧抽屉滑出 → 展示统计+口语录音+AI 报告
6. 导航到 `/teacher/messages` → 展示师生聊天界面
7. 导航到 `/teacher/assistant` → 发送"分析班级整体学情" → 展示回复
