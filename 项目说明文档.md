# RealEnglish（真语英语）项目说明文档

> 武汉理工大学"华工杯"AI 应用创新大赛参赛作品 · AI+开发赛道

---

## 一、项目概述

### 1.1 作品名称

**RealEnglish（真语英语）—— AI 驱动的兴趣激发式英语视听说教练**

### 1.2 一句话简介

基于真实影视片段，通过 DeepSeek 大模型智能出题、科大讯飞语音评测、LangGraph 多智能体协同调度，打造"看→听→说→评→管"五步闭环的沉浸式英语学习平台。

### 1.3 目标用户

- **学生端**：大学生英语学习者，通过影视兴趣驱动提升听说能力
- **教师端**：英语教师，高效管理班级、追踪学情、精准辅导

---

## 二、需求背景

### 2.1 传统英语学习的痛点

1. **素材枯燥**：教材对话脱离真实语境，学生缺乏学习动力
2. **反馈滞后**：口语练习缺乏即时评测，学生不知道发音是否准确
3. **个性化不足**：统一进度无法适配不同水平的学生
4. **教师负担重**：大班教学中教师难以逐一追踪学情，预警滞后

### 2.2 AI 赋能的解决方案

| 痛点 | RealEnglish 方案 |
|------|-----------------|
| 素材枯燥 | 嵌入 B 站真实影视片段，对话文本中英对照，兴趣驱动 |
| 口语缺乏评测 | 科大讯飞 ISE 引擎实时评分（准确度/流畅度/完整度/总分），AI Coach 生成激励反馈 |
| 个性化不足 | DeepSeek 大模型根据对话内容 + 学生等级自适应出题，定级测试智能分级（A/B/C） |
| 教师管理难 | LangGraph 多智能体自动生成学情报告、学习预警、班级周报，16+ 工具辅助教学决策 |

## 四、实现细节

### 4.1 智能出题

**流程**：片段对话文本 → DeepSeek 分析 → 托业/四六级风格听力题（Part2 问答/Part3 对话理解）

**LangGraph 工作流**：`quiz_app`（双节点：generate → format），自动校验 JSON 格式、缓存已生成题目

**难度自适应**：根据学生等级 A/B/C 调整题目难度，定级测试含 10 道混合难度题

### 4.2 口语评测

**架构**：
```
MediaRecorder(WebM/Opus) → ffmpeg 转码(PCM 16kHz) → 讯飞 ISE WebSocket → XML 解析 → 4 维评分
                                                                              ↘ DeepSeek → AI 激励反馈
```

**评分维度**：total_score / accuracy / fluency / integrity（5 分制）

**降级策略**：讯飞 API 不可用时返回默认评分 + 通用鼓励语

### 4.3 多智能体系统（LangGraph）

**技术特点**：
- ReAct Agent 模式（推理→调用工具→观察→继续推理）
- SqliteSaver checkpointer 持久化会话状态
- SSE 流式输出（JSON 行事件），前端实时渲染工具调用状态
- LangSmith 可追踪完整循环

### 4.4 单词本 + 错题本（"我的积累"）

- **查词**：点击对话中的英文单词 → DeepSeek 返回音标/释义/例句 → 收藏
- **释义存储**：收藏时同步存入 definition，列表即时显示
- **复习模式**：跳动柱状遮罩遮掩释义，点击显示
- **检测模式**：DeepSeek 根据单词本生成选择题，答错自动入错题库 + 标为未掌握
- **进度环**：SVG 圆环动画展示掌握率

### 4.5 学习预警

教师看板自动标记三类学生：
| 类型 | 规则 | 颜色 |
|------|------|------|
| 未定级 | level IS NULL | 灰色 |
| 不活跃 | 近 7 天无练习 | 橙色 |
| 正确率骤降 | 本周比上周下降 ≥30% | 红色 |

### 4.6 情景式自由对话

AI 生成生活场景（餐厅点餐、机场登机等）和角色设定，用户与 AI 进行多轮英语对话（支持文字/语音输入），结束后四维度统一评测。

### 4.7 双工作台分离

学生端和教师端共享同一应用，通过独立路由树 + 独立布局 + 独立导航完全分离，登录后按角色自动跳转。

## 五、创新点

### 5.1 真实影视驱动的兴趣激发

摒弃枯燥教材，以 B 站美剧/英剧/电影片段为载体。学生选择感兴趣的内容 → 观看片段 → AI 针对该片段出题 → 跟读台词 → 形成"兴趣→练习→反馈"闭环。

### 5.2 工具调用型学习/教学 Agent

不是简单的"聊天机器人"，而是**真正能做事**的 ReAct Agent：
- 学生说"今天该复习什么"→ Agent 调用 `get_smart_review_plan`，综合单词薄弱点+高频错题+任务进度，给出具体到单词/题号的复习清单
- 教师说"公选1班学情报告"→ Agent 调用 `generate_group_report`，一次 LLM 调用生成完整班级报告（无需逐学生拼接）

### 5.3 流式大模型翻译

对话区的"大模型翻译"按钮**按需触发**流式翻译（而非预先缓存），逐 token 输出到中文区域，翻译结果写入 localStorage 跨会话持久化，支持"重新生成"。

### 5.4 AI 出题 × 单词本联动

单词本不仅是收藏夹，**"检测模式"**让 DeepSeek 根据用户真实单词库生成个性化选择题，答错自动联动错题本，形成"学单词→检测→纠错"闭环。

### 5.5 多维度学习预警

不仅看"学了多少"，更看"哪里退步了"。正确率骤降检测通过本周 vs 上周对比，自动识别学习曲线拐点。

### 5.6 邀请码绑定

教师生成 4 位字母数字邀请码，学生注册时输入即自动绑定班级，告别传统手动分配，简化大班管理。
